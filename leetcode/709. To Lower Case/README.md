## Solution 1.

```cpp
// OJ: https://leetcode.com/problems/to-lower-case/
// Author: github.com/punkfulw
// Time: O(N)
// Space: O(1)
class Solution {
public:
    string toLowerCase(string str) {
        for (char &c : str) c = tolower(c);
        return str;
    }
};
```
